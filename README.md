# Contact Manager

A responsive **frontend contact management application** built with **React** and **Tailwind CSS**, using **JSON Server** as a mock REST API to demonstrate real-world CRUD workflows and API integration.

This project focuses on clean component structure, API-driven state management, and modern frontend best practices.
